# Lesson 01 — HTML基礎

> **作業フォルダ**: `exercises/lesson01/`

---

## HTMLとは？

HTML（HyperText Markup Language）は、Webページの **構造** を作る言語です。
「ここは見出し」「ここは段落」「ここはリンク」といった **役割** をブラウザに伝えます。

---

## Step 1 — HTMLの基本構造

`exercises/lesson01/index.html` を作成し、以下を入力してください。
（環境構築ガイドで作成済みの場合は、中身を書き換えてください）

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>HTML基礎レッスン</title>
</head>
<body>

  <h1>HTMLの基礎を学ぼう</h1>

</body>
</html>
```

保存して Live Server で確認しましょう。見出しが表示されればOKです。

### 各行の意味

| コード | 意味 |
|--------|------|
| `<!DOCTYPE html>` | 「これはHTMLの文書です」という宣言 |
| `<html lang="ja">` | HTML全体を囲むタグ。`ja` は日本語ページの意味 |
| `<head>` | ページの設定情報（ブラウザには表示されない） |
| `<meta charset="UTF-8">` | 日本語を正しく表示するための設定 |
| `<title>` | ブラウザのタブに表示されるタイトル |
| `<body>` | **ブラウザに表示される中身**（ここに書いていく） |

> **覚えておくこと**: タグには `<開始タグ>` と `</閉じタグ>` のペアがあります。
> `<h1>` で始めたら `</h1>` で閉じます。

## Step 2 — 見出しと段落

`<body>` の中身を以下のように書き換えてみましょう。

```html
<body>

  <h1>自己紹介ページ</h1>

  <h2>はじめまして</h2>
  <p>私の名前は田中太郎です。Web制作の勉強をしています。</p>
  <p>趣味は読書とゲームです。</p>

  <h2>好きな食べ物</h2>
  <p>カレーライスが一番好きです。</p>

</body>
```

### ポイント

| タグ | 意味 | 見た目 |
|------|------|--------|
| `<h1>` | 一番大きい見出し（ページに1つ） | 大きく太い文字 |
| `<h2>` | 2番目の見出し | やや大きい文字 |
| `<p>` | 段落（Paragraph） | 普通の文字、前後に余白 |

> **ルール**: `<h1>` はページに1つだけ。見出しは `h1` → `h2` → `h3` の順に使います。

### やってみよう

`<h1>` の名前を自分の名前（架空でOK）に変えて、自己紹介文を書いてみましょう。

## Step 3 — リスト

段落の下に、リストを追加してみましょう。

### 箇条書き（順序なしリスト）

```html
<h2>趣味リスト</h2>
<ul>
  <li>読書</li>
  <li>ゲーム</li>
  <li>映画鑑賞</li>
</ul>
```

### 番号付きリスト（順序ありリスト）

```html
<h2>好きな食べ物ランキング</h2>
<ol>
  <li>カレーライス</li>
  <li>お寿司</li>
  <li>ラーメン</li>
</ol>
```

| タグ | 意味 |
|------|------|
| `<ul>` | 箇条書きリスト（・で始まる） |
| `<ol>` | 番号付きリスト（1, 2, 3...） |
| `<li>` | リストの各項目 |

### やってみよう

自分の趣味や好きなものをリストにして追加してみましょう。

## Step 4 — リンクと画像

### リンク

```html
<h2>リンク</h2>
<p><a href="https://www.google.com">Googleを開く</a></p>
```

| 属性 | 意味 |
|------|------|
| `href` | リンク先のURL |

> `<a>` タグの中に書いたテキストがクリックできるリンクになります。

### 画像

```html
<h2>画像</h2>
<img src="https://via.placeholder.com/300x200" alt="サンプル画像">
```

| 属性 | 意味 |
|------|------|
| `src` | 画像のURL（またはファイルパス） |
| `alt` | 画像が表示できないときの代わりのテキスト |
| `width` | 画像の幅（例：`width="300"`） |

> **重要**: `<img>` には閉じタグがありません。`alt` 属性は必ず書きましょう。

### やってみよう

好きなWebサイトへのリンクを1つ追加してみましょう。

## Step 5 — div でグループ化

HTMLでは、いくつかの要素をまとめてグループにしたいことがあります。
`<div>` タグを使うと、要素をグループ化できます。

```html
<div>
  <h2>プロフィール</h2>
  <p>田中太郎 / 20歳</p>
  <p>東京都在住</p>
</div>

<div>
  <h2>趣味</h2>
  <ul>
    <li>読書</li>
    <li>ゲーム</li>
  </ul>
</div>
```

`<div>` 自体は見た目に変化はありません。
次のレッスンで学ぶ CSS を使うときに、「このグループだけ背景色を変える」といった使い方をします。

> **発展**: HTMLには `<header>`、`<main>`、`<footer>` など、意味を持つグループ化タグもあります。
> 今はまず `<div>` を使えるようになりましょう。
