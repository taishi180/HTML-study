## Step 2 — まずHTMLから作ろう

`index.html` を開いてください。コメントで①〜⑥の指示が書いてあります。
**上から順番に**、各コメントの下にHTMLを書いていきましょう。

### ① ヘッダーを作る

ロゴ画像とナビゲーションリンクを並べます。

```html
<div class="header">
  <img src="img/logo.png" alt="TeamOCAロゴ" width="80">
  <div class="header-nav">
    <a href="#">news</a>
    <a href="#">schedule</a>
    <a href="#">shop</a>
  </div>
</div>
```

> Lesson 01 で学んだ `<img>` と `<a>` を使っています。

### ② Aboutセクションを作る

チーム名と説明文を書きます。

```html
<div class="about">
  <h1>TeamOCA</h1>
  <p>チームOCAはゲームが大好きなメンバーが集まったチームです。一緒にゲームを楽しみましょう！</p>
</div>
```

### ③ メンバーセクションを作る

メンバーの画像と名前を並べます。まず1人分の書き方を覚えましょう。

```html
<div class="section">
  <h2>MEMBER</h2>
  <div class="member-list">

    <div class="member">
      <img src="img/member/member_01.jpeg" alt="メンバー1">
      <p>member 1</p>
    </div>

    <!-- ここに member_02, member_03, member_04 を同じ形で追加しよう -->

  </div>
</div>
```

> member_01 の部分をコピーして、数字を 02, 03, 04 に変えるだけ！

### ④ ゲームセクションを作る

③と同じ構造で、クラス名を変えるだけです。

```html
<div class="section">
  <h2>GAME TITLE</h2>
  <div class="game-list">

    <div class="game">
      <img src="img/game/game_01.jpeg" alt="ゲーム1">
      <p>GAME 1</p>
    </div>

    <!-- game_02 も追加しよう -->

  </div>
</div>
```

### ⑤ お問い合わせセクション

```html
<div class="section">
  <h2>CONTACT</h2>
  <p>お問い合わせは teamoca@example.com までご連絡ください。</p>
</div>
```

### ⑥ フッター

```html
<div class="footer">
  <p>&copy; 2026 TeamOCA</p>
</div>
```

> `&copy;` は &copy; マークになる特殊な書き方です。

## Step 4 — ヘッダーのスタイル

ロゴとナビを Flexbox で「両端に配置」します。

```css
/* ===== ヘッダー ===== */
.header {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 16px 40px;
  background-color: #1a2a3a;
}

.header-nav {
  display: flex;
  gap: 24px;
}

.header-nav a {
  color: #ccc;
  text-decoration: none;
  font-size: 14px;
}
```

> Lesson 03 で学んだ `display: flex` + `justify-content: space-between` を使っています！

## Step 5 — Aboutセクションのスタイル

```css
/* ===== About ===== */
.about {
  text-align: center;
  padding: 60px 40px;
}

.about h1 {
  font-size: 36px;
  margin-bottom: 16px;
}

.about p {
  color: #aaa;
  max-width: 600px;
  margin: 0 auto;
}
```

> `margin: 0 auto` は「左右の余白を自動にする」= 中央寄せの定番テクニックです。

## Step 7 — メンバーセクション（Flexbox で横並び）

ここが一番の見どころです！

```css
/* ===== メンバー ===== */
.member-list {
  display: flex;
  justify-content: center;
  gap: 24px;
  flex-wrap: wrap;
}

.member {
  text-align: center;
  width: 150px;
}

.member img {
  width: 150px;
  height: 150px;
  border-radius: 50%;
  object-fit: cover;
}

.member p {
  margin-top: 8px;
  font-size: 14px;
  color: #ccc;
}
```

> `border-radius: 50%` で画像が丸くなります！
> `object-fit: cover` は画像が歪まないようにする設定です。
