# 番外編 — パスの書き方（相対パスと絶対パス）

HTMLで画像を表示したり、CSSファイルを読み込むとき、**ファイルの場所（パス）** を指定します。
このパスの書き方には2種類あります。

---

## 絶対パス

ファイルの場所を **最初から全部** 書く方法です。

```html
<!-- Web上の画像（URLで指定） -->
<img src="https://example.com/img/photo.jpg" alt="写真">

<!-- パソコン内のファイル（フルパスで指定） -->
<img src="C:/Users/Owner/Documents/HTMLwork/img/logo.png" alt="ロゴ">
```

### 特徴

- どこから見ても同じ場所を指す
- Web上の画像（URL）は絶対パスで書く
- パソコン内のフルパスは **他の人の環境では動かない** ので基本使わない

---

## 相対パス

**今いるファイルの場所を基準に** 書く方法です。
HTMLやCSSでは **こちらをよく使います。**

---

### 同じフォルダ内のファイル

```
lesson01/
├── index.html    ← 今ここ
└── style.css
```

```html
<!-- index.html から style.css を読み込む -->
<link rel="stylesheet" href="style.css">
```

同じフォルダにあるファイルは、**ファイル名だけ** でOKです。

---

### 下のフォルダ（子フォルダ）にあるファイル

```
lesson04/
├── index.html    ← 今ここ
└── img/
    └── logo.png
```

```html
<!-- index.html から img フォルダ内の logo.png を指定 -->
<img src="img/logo.png" alt="ロゴ">
```

`フォルダ名/ファイル名` のように **`/` で区切って** 書きます。

---

### さらに深いフォルダ

```
lesson04/
├── index.html    ← 今ここ
└── img/
    └── member/
        └── member_01.jpeg
```

```html
<img src="img/member/member_01.jpeg" alt="メンバー1">
```

フォルダが深くなっても、`/` でつないでいくだけです。

---

### 上のフォルダ（親フォルダ）にあるファイル

```
project/
├── css/
│   └── style.css    ← 今ここ
└── img/
    └── logo.png
```

```css
/* style.css から logo.png を指定 */
background-image: url(../img/logo.png);
```

**`..`（ドット2つ）** で「1つ上のフォルダに戻る」という意味になります。

| 書き方 | 意味 |
|--------|------|
| `./` または省略 | 今いるフォルダ（同じ階層） |
| `../` | 1つ上のフォルダに戻る |
| `../../` | 2つ上のフォルダに戻る |

---

## よくある例

### Lesson 04 の場合

```
lesson04/
├── index.html
├── style.css
└── img/
    ├── logo.png
    ├── member/
    │   └── member_01.jpeg
    └── game/
        └── game_01.jpeg
```

`index.html` から見たパス：

```html
<!-- CSSの読み込み（同じフォルダ） -->
<link rel="stylesheet" href="style.css">

<!-- ロゴ画像（img フォルダの中） -->
<img src="img/logo.png" alt="ロゴ">

<!-- メンバー画像（img → member フォルダの中） -->
<img src="img/member/member_01.jpeg" alt="メンバー1">

<!-- ゲーム画像（img → game フォルダの中） -->
<img src="img/game/game_01.jpeg" alt="ゲーム1">
```

---

## まとめ

| 種類 | 書き方 | 使う場面 |
|------|--------|---------|
| 絶対パス | `https://example.com/img/photo.jpg` | 外部サイトの画像やリンク |
| 相対パス | `img/logo.png` | 自分のプロジェクト内のファイル |

> **基本は相対パスを使おう。**
> 自分のプロジェクト内のファイル（画像、CSS、他のHTML）は相対パスで書くのが基本です。
> 外部サイトのURL（`https://...`）だけ絶対パスを使います。
