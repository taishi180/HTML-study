# 環境構築ガイド — VSCode ＋ Live Server

## このカリキュラムの全体像

このカリキュラムでは、4つのレッスンを通じてHTML＆CSSの基礎を学びます。
最後にはチームのWebサイトを1ページ作れるようになります。

| レッスン | 内容 | ゴール |
|---------|------|--------|
| 01 | HTML基礎 | Webページの骨組みを作れる |
| 02 | CSS基礎 | ページに色やデザインをつけられる |
| 03 | Flexbox | 要素を横並びにできる |
| 04 | 総合課題 | チームのWebサイトを作る |

---

## 1. VSCode をインストールする

すでにインストール済みの場合はスキップしてください。

1. [https://code.visualstudio.com/](https://code.visualstudio.com/) にアクセス
2. 「Download for Windows」をクリック
3. ダウンロードしたインストーラを実行
4. インストール中の設定はすべてデフォルトでOK（「PATHに追加する」にはチェックを入れる）

---

## 2. 日本語化（任意）

1. VSCode を開く
2. 左サイドバーの **拡張機能アイコン**（四角が4つ並んだマーク）をクリック

## 3. Live Server 拡張機能をインストールする

Live Server を使うと、HTMLファイルを保存するたびにブラウザが自動でリロードされます。

1. 左サイドバーの **拡張機能アイコン** をクリック
2. 検索欄に `Live Server` と入力
3. 作者が **Ritwick Dey** の「Live Server」を選び **Install** をクリック

---

## 4. 作業フォルダを開く

1. VSCode のメニュー → **ファイル → フォルダーを開く**
2. `Documents/HTMLwork` フォルダを選択して「フォルダーの選択」
3. 左サイドバーの **エクスプローラー** にフォルダの中身が表示される

> **ポイント**: フォルダを開いた状態で作業することが大切です。
> ファイル単体で開くと Live Server がうまく動かないことがあります。

---

## 5. HTMLファイルを作成して Live Server で表示する

### ファイルを作る

1. エクスプローラーの `exercises/lesson01` フォルダを右クリック → **新しいファイル**
2. ファイル名を `index.html` と入力して Enter

### HTMLの雛形を入力する

作成した `index.html` を開き、以下をそのまま入力してください。

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>はじめてのHTML</title>
</head>
<body>
  <h1>Hello, World!</h1>
  <p>ブラウザに表示されたら成功です！</p>
</body>
</html>
```

> **Emmet ショートカット**
> ファイルの1行目で `!`（半角の感嘆符）を入力してから `Tab` キーを押すと、
> HTMLの雛形が自動で展開されます。毎回手打ちしなくて済むのでとても便利です。
> うまくいかない場合は `!` を入力した後に出る候補を `Enter` で選んでもOKです。

### Live Server で表示する

方法は2つあります。どちらでもOKです。

**方法A — ステータスバーから起動**
- VSCode 画面の右下に表示されている **「Go Live」** をクリック

**方法B — 右クリックから起動**
- エディタ内で右クリック → **「Open with Live Server」** を選択

ブラウザが自動で開き「Hello, World!」が表示されれば成功です！

## 6. 作業の流れ

これからの学習では、以下の流れを繰り返します。

```
① ガイドを読む（この guides フォルダ内のファイル）
② exercises フォルダ内に HTML/CSS ファイルを作成・編集する
③ 保存する（Ctrl + S）
④ ブラウザが自動で更新される → 結果を確認
```

> **よく使うショートカット**
> | 操作 | ショートカット |
> |------|-------------|
> | ファイル保存 | `Ctrl + S` |
> | 元に戻す | `Ctrl + Z` |
> | やり直し | `Ctrl + Shift + Z` |
> | コピー | `Ctrl + C` |
> | ペースト | `Ctrl + V` |
> | 行を複製 | `Alt + Shift + ↓` |
> | コメントアウト | `Ctrl + /` |
